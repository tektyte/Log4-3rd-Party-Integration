Log4 USB and POE Serial Output

Takes in serial information from Log4 USB or POE device then
displays measurements on RaspberryPi python terminal.

This example code is in the public domain.

Units supported
 Current - "uA", "mA", "A"
 Voltage - "mV", "V"
 Power - "mW", "W"

Dependencies
-pyserial
